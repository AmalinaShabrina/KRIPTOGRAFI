CARA MENJALANKAN PROGRAM
1. install tkinter : pip install tkinter (jika belum install tkinter)
2. masuk ke file .py yang diinginkan
3. run program melalui terminal dengan C:/Users/USER/AppData/Local/Programs/Python/Python312/python.exe c:/Users/USER/kriptografi/<nama file>
4. Masukkan teks pada kolom input atau buka file berformat .txt dengan menekan tombol "Upload File"
5. Masukkan kunci minimal 12 karakter
6. Klik tombol "Encrypt" untuk mengenkripsi dan klik tombok "Decrypt" untuk mendeskripsi
7. Hasil akan muncul
